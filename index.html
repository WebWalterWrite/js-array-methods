Les principales méthodes utilisées pour exploiter un Array en javascript. Prenons en exemple un tableau contenant des noms de villes US.

Array.map

Afficher toutes les Villes

Que fait array.map ?

Retourne un nouvel array traité par notre fonction passée en callback. Idéalement utilisé pour lire chaque entrée d'un tableau et y effectuer un traitement.

Prenons en exemple, nous récupérons depuis une api ou notre base de données une liste de villes sous forme de tabeau

            const UsCity = [ "New York", "Miami", "Los Angeles", "Chicago", "Washington"]; // Tableau city

A présent nous souhaitons afficher la liste des villes dans notre document.

Pour ce faire nous déclarons une fonction nommée AllCities qui aura en son corps les instructions suivantes :

  • Séléctionner l'élément ayant le id -_allcity- qui contiendra notre liste et le stocker dans la variable -content- .
  • Lorsque la fonction est invoquée, la variable content est vidée de son contenu html.
              const AllCities = () => {

                 let content = document.querySelector("#_all_cities");
        
                 content.innerHTML = "";
              
                 UsCity.map( el => content.innerHTML += `<li>${el}</li>` );
              };

    Array.fill

    Classement Rank Stars

    Cette méthode permet de modifier intégralement ou partiellement les entrées d'un tableau

    Voyons de plus près la syntaxe :

    Pour l'exemple nous souhaitons donner une note entre et 1 et 5 étoiles à notre langage préféré qui est le Javascript

    je crée dans un 1er temps un tableau contenant notre . ( Pour ce faire j'utilise FontAwesome)

                    const stars = '<span><i class="far fa-star fa-3x"></span>';

                const arrStars = [...Array(5)].fill(stars);

    Une fois ceci fait, lorsque l'on cliquera sur une des 5 étoiles, la fonction fillStars remplira le tableau selon la position cliquée.

    Les arguments à transmettre à notre fonction :

    • -i- correspond à l'index de notre array arrStars lors du clic
    • -array- correspond à notre array arrayStars

    Cette dernière créée d'abord notre étoile.

                    const fillStars = ( i, array ) => {
                
                  const fullStars = '<span><i class="fas fa-star fa-3x"></i></span>';
                
                  array.fill(fullStars,0, i )
                
                  content.innerHTML = array.join('');
                
                }
